# Hackintosh-F555LB-Mojave
本项目由基于[https://github.com/emre1393/asus-x555lb-mojave-efi](https://github.com/emre1393/asus-x555lb-mojave-efi)
在此基础上适配了10.14.6系统在F555LB电脑上

# 配置
CPU i5-5200U
GPU 940M
分辨率 1366x768
声卡 ALC233


# 正常
核显
声卡（耳机使用时调到全右，否则只输出背景音无人声）
有线网卡
键盘和Fn按键（亮度快捷键不可用）
触控板
亮度调节
亮度调节保存
电量显示
USB3.0和USB2.0

# 不正常
940M显卡（无解）
Wifi（更换白果网卡）
蓝牙（更换白果网卡）
照相机（没弄）
读卡器（没弄）


# 未测试
HDMI 图像
HDMI 音频
VGA 端口
